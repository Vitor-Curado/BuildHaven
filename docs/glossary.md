# 📖 Glossary

- **Askama** — Rust server-side templating engine
- **Axum** — Rust web framework
- **Tokio** — async runtime
- **Serde** — serialization/deserialization library
- **SQLx** — Rust SQL toolkit
- **Podman** — container engine (Docker alternative)
- **GHCR** — GitHub Container Registry
- **tower-http** — HTTP middleware utilities
- **tower-governor** — rate limiting middleware
- **dotenvy** — environment variable loader
- **Distroless** — minimal container runtime image
- **systemd** — Linux service manager
- **Nginx** — reverse proxy server


This is one of the most valuable additions you can make.
